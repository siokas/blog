# Code Complexity Measurement

The creator of Laravel PHP Framework, Taylor Otwell, a few days ago published in his new Medium blog a very interesting statistic. There is a nice tool that helps developers to compare their projects with other similar projects regarding the complexity of their code. In this particular measurement, Taylor, measured Laravel with other PHP Frameworks such as Symfony, Slim, Cake and Zend.

# Results

The results were so remarkable about Laravel. Laravel has the shortest average of lines of code per method and an extraordinary average method complexity of 1.62. Just to mention the difference, the second framework in the average method complexity was Slim with average of 2.40. (Of course the lower the better)
